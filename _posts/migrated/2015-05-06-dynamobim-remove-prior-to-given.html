--- layout: post title: DynamoBIM- Remove Prior to Given Character date: '2015-05-06T09:51:00.001-07:00' author: JohnP tags: modified_time: '2015-05-06T09:52:51.134-07:00' thumbnail: http://lh3.googleusercontent.com/-qDGffAqNAi4/VUpGkmj8MtI/AAAAAAAAAMk/ngHwcTnj1mQ/s72-c/2015-05-06_10h48_30_thumb.png?imgmax=800 blogger_id: tag:blogger.com,1999:blog-8275462422374771907.post-7388961541023946263 blogger_orig_url: http://sixtysecondrevit.blogspot.com/2015/05/dynamobim-remove-prior-to-given.html --- A question came up over at the DynamoBIM forums regarding how to remove all characters prior to a given character.
Enter String.RemovePriorToCharacter
This custom node takes the input and searches for the given character. It then removes everything from that character and before. *Note it finds the first instance of the given character and removes from there.
Get it on the package manager now.
